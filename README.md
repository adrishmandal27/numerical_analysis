PROJECT TITLE:-  Numerical Methods: From Theory to Implementation
ABOUT:-  Bridging between Numerical Analysis and Computer Coding
PURPOSE:-    A collection of numerical methods and algorithms implemented for solving mathematical problems that are dfficult or impossible to solve analytically. This repository serves as a practical toolkit for topics including root-finding, interpolation, solving systems of linear equations, numerical differentiation, and numerical integration.
    1. Root-Finding Methods
        a. Bisection Method
        b. Regula Falsi Method
        c. Newton Raphson Method
        d. secant Method
    2. Interpolation & Approximation
        a. Lagrange Interpolation
        b. Newton Divide Interpolation
        c. Newton Forward Divide Difference Method
        d. Newton Backward Divide Difference Method
    3. Linear Algebra & Systems
        a. Gauss Jacobi Method
        b. Gauss Seidel Method
    4. Numerical Differentiation
        a. Differentiation using Newton Forward Divide Difference
        b. Differentiation using Newton Backward Divide Difference
    5. Numerical Integration
        a. Trapezoidal Rule
        b. Simpson's 1/3rd Rule
        c. Simpson's 3/8th Rule
        d. Gauss_Legendre's Method

Note**:-    99% of the work is dome by the author and 1% with the help of AI.
